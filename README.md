<h1 align="left">River Crossing Riddle</h1>

###

<p align="left">This project is an MVC web version of the classic river crossing logic puzzle. This project was made using dotnet framework 4.7.2.<br><br>This version features digitally drawn pixel art by <a href="https://github.com/Molly266" target="_blank">Molly266</a> for both the characters and background,<br><br>To win, all characters must make it to the other side of the river!<br>The rules of the game are:<br><br>• Only the farmer can row the boat.<br>• The chicken must not be left alone with the corn or the chicken will eat the corn.<br>• The fox must not be left alone with the alone or the fox will eat the chicken.</p>

###

<h3 align="left">Features</h3>

###

<p align="left">• Interactive web-based gameplay<br>• Validation of player moves with appropriate messages displayed<br>• Clear win/lose conditions and messages presented to player</p>

###

<h3 align="left">Languages / Libraries Used</h3>

###

<p align="left">• C#<br>• JavaScript<br>• HTML/CSS<br>• jQuery</p>
